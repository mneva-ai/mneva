# Mneva

**Persistent agent context substrate.**

A local-first CLI that stores your knowledge as markdown files so any AI tool can query the same context through a localhost API.

> Lossless, frictionless context reconstruction across sessions and tools.

This is a **placeholder release (v0.0.1)** to reserve the package name. The real CLI is under active development.

- Website: https://mneva.org
- Repository: https://github.com/mneva-ai/mneva
- License: Apache 2.0

## Design Principles

- **Explicit** — context is visible, inspectable, editable
- **Yours** — data lives on your machine in plain markdown
- **File over app** — open format, no lock-in
- **BYOAI** — bring your own AI keys, choose your own models
